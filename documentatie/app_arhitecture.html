Scop

Scopul aplicatiei TraST este de a oferi continut pentru invatarea aspectelor teoretice pentru examenul teoretic din cadrul obtinerii permisului de conducere.

Tehnologiile folosite

HTML si CSS

HTML

HTML a fost folosit pentru crearea componentelor si a continutului ce face parte din interfeta cu care interactioneaza utilizatorii.

Fiind limbajul standard pentru crearea aplicatiilor web, folosirea lui a fost necesara.

In decursul dezvoltarii proiectului, am invatat multe alte noi concepte despre HTML, si am aprofundat conceptele deja stiute.

CSS

Limbajul CSS a fost folosit pentru stilizarea si imbunatairea aspectului de scalare a site-ului in functie de device-ul de pe care este accesat acesta(responsive).

Avand in vedere ca fata de HTML, CSS este folosit pentru a specifica fiecare componenta descrisa in HTML, acest limbaj este necesar in cadrul oricarei aplicatii Web de la care ne asteptam sa aiba un anumit aspect.

Prin modularizarea codului CSS si impartirea acestuia in diferite fisieare .css, fata de varianta de a avea tot codul intrun fisier .css unic, am reusit sa organizam mai bine aspectele de styling ale site-ului.

JavaScript

Javascript este principalul limbaj de programare pentru crearea unor pagini Web dinamice, deci este prezent in marea majoritate a aplicatiilor Web. Prin JavaScript am reusit sa implementam diferite aspecte din cadrul aplicatiei TraST, in mod special prin suita de tehnologii AJAX.

De-a lungul proiectului am intampinat diferite dificultati in implementarea codului, dar intr-un final, am reusit sa rezolvam problemele.

AJAX(Asynchronous JavaScript And XML)

Principala functionalitate a limbajului JavaScript pe care am folosit-o in cadrul aplicatiei este suita de tehnologii AJAX.

Prin folosirea AJAX-ului am reusit sa creem pagini web dinamice, facand request-uri catre server prin script-uri PHP, si afisand raspunsurile pe partea de front-end a aplicatiei folosind JavaScript.

PHP

Limbajul de programare PHP este limbajul folosit atat pentru validarea si obtinerea datelor din baza de date, dar si pentru prelucrarea datelor si afisarea acestora in anumite componente ale aplicatiei

Fiind un limbaj de programare creat special pentru web-development, fiecare aspect al lui este de asemenea orientat pe partea de web, fiind foarte bine documentat. Orice problema am intampinat, am gasit mereu o solutie.
